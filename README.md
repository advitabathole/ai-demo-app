# AI Demo App (Streamlit)

A simple interactive AI demo app built with Streamlit.  
This app performs sentiment analysis on user-entered text using a trained ML model.

---

## Features
- Clean UI built with Streamlit  
- Loads a pretrained NLP model  
- Classifies text sentiment in real-time  
- Great starter AI web app project  

---

## How It Works
- User enters text
- Text is vectorized using TF-IDF
- Logistic Regression model predicts sentiment
- Result is displayed in the app
